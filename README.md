# 🌿 PreLeave

A full-stack Employee Leave Management System built using **Node.js, Express.js, PostgreSQL, and EJS**.

The application provides separate portals for **HR Administrators** and **Employees**, allowing employee management, leave applications, attendance tracking, and leave approval workflows.

## ✨ Features

### HR Portal
- Employee CRUD operations
- Leave request approval/rejection
- Attendance dashboard
- Employee profile management
- CSV report export
- Dashboard analytics

### Employee Portal
- Apply for leave
- View leave history
- Leave balance dashboard
- Employee profile
- Live leave status updates

## 🛠 Tech Stack

- Node.js
- Express.js
- PostgreSQL
- EJS
- HTML/CSS/JavaScript
- Multer
- bcrypt

## 🏗 Architecture

- MVC Architecture
- Express Routers
- PostgreSQL Database
- Dynamic EJS Rendering
- RESTful CRUD Operations

## 🗄 Database

Main tables:

- Employees
- Leave Requests
- Attendance

## 🔐 Security

- Password hashing using bcrypt
- Parameterized SQL queries
- Environment variables for database credentials
